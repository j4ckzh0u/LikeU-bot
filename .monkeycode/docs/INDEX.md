# AI 研发助手 - 项目概述

## 1. 项目简介

### 项目名称
**AI 研发助手** (AI Coding Assistant)

### 项目定位
企业级 AI 开发平台，覆盖需求 → 设计 → 开发 → 代码审查全流程。

### 核心目标
通过自然语言交互，让 AI 完成从需求分析、技术设计、代码开发到代码审查的完整开发流程。

### 目标用户
- 研发团队（开发人员、测试人员）
- 技术管理者（项目经理、架构师）
- 需要提升研发效率的企业团队

---

## 2. 核心功能

### 2.1 智能任务（核心）
用自然语言描述需求，AI 自动完成开发、设计或代码审查。

**功能特性**：
- 自然语言任务描述
- 支持多种 AI 模型（OpenAI GPT、Claude、国产大模型等）
- Git 仓库关联与代码导入
- 代码生成与优化
- 技术文档自动生成

### 2.2 项目管理
关联 Git 仓库，管理项目需求和任务。

**功能特性**：
- 需求文档管理
- 设计任务创建与跟踪
- 开发任务分配
- 进度追踪与统计

### 2.3 在线开发环境
提供完整的在线开发环境。

**功能特性**：
- **在线 IDE**：支持多语言语法高亮的代码编辑器
- **Web 终端**：支持多会话的浏览器终端
- **文件管理**：在线浏览、编辑、上传、下载文件
- **在线预览**：一键预览 Web 服务运行效果
- **远程协助**：支持与他人共享终端会话

### 2.4 代码审查
配置 Git 机器人，自动审查 GitHub / GitLab / Gitee 的 PR/MR。

**功能特性**：
- 自动拉取代码变更
- 智能代码分析
- 审查意见生成
- 改进建议提出

### 2.5 团队协作
团队管理员可管理成员、分配资源。

**功能特性**：
- 团队成员管理
- 资源分配（宿主机、镜像、AI 模型）
- 权限控制
- 资源使用统计

## 3. 技术架构

### 3.1 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 19 + TypeScript + Vite + Tailwind CSS |
| 后端 | Go + Echo + Ent ORM |
| 数据库 | PostgreSQL 16 + Redis 7 |
| 桌面端 | Electron 35 |
| 移动端 | Capacitor 7（Android / iOS） |

### 3.2 项目结构

```
ai-coding-assistant/
├── frontend/          # Web 前端（React + TypeScript）
├── backend/           # 后端服务（Go）
├── desktop/           # 桌面客户端（Electron）
└── mobile/            # 移动客户端（Capacitor）
```

### 3.3 部署架构

**私有化部署**：
- 支持部署到企业自己的服务器
- 支持 Docker Compose 一键部署
- 支持 Kubernetes 集群部署

---

## 4. AI 模型支持

### 支持的模型类型
- OpenAI GPT 系列（GPT-4o、GPT-4o-mini 等）
- Anthropic Claude 系列（Claude 3.5 Sonnet 等）
- 国产大模型（通义千问、智谱 GLM 等）

### 模型配置
- 支持多模型切换
- 支持自定义模型 Endpoint
- 支持模型配额管理
